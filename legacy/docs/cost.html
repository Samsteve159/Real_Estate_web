06 Worth knowing

The spend cap is your safety net. A hard monthly limit is set on the AI account, so the cost can never run past whatever ceiling you choose — regardless of how busy the tools get.
  • Hybrid B (Haiku chat) is the cheapest by a wide margin and is genuinely capable for a listings chatbot — roughly half the all-in cost of Hybrid A.
  • Hybrid A (Sonnet chat) buys a little extra polish in conversation for a modest step up in cost.
  • Both keep Opus on the valuation, so the number a seller sees is always produced by the most capable model.
  • "Prompt caching" can be switched on later to shave the chat cost further — the listings data is re-sent on every message and could be cached.
  • Note: using anything below Opus is a deliberate choice — the current standard is "Opus 4.8 or above", so this is the owner's call.
How to read these numbers. All figures are estimates, not quotes. They scale with how many people use the tools and how long each chat runs, so treat them as a budgeting range rather than a fixed bill. USD converted to AUD at roughly 1.5×. Hosting and SMS prices vary by provider.
