# Assets — drop your files here

Paste the source files for the new build into the matching folder below.
Claude reads from here; nothing in this folder is published as-is.

| Folder | What goes here | Used for |
|---|---|---|
| `logo/` | The **new Manifest logo** (SVG preferred; PNG/JPG fine). Add light + dark versions if you have them. | Site header, favicon, newsletter, app |
| `brand/` | Anything brand-related: colour codes, fonts, an existing style guide, business cards, signage photos. | Building the design system |
| `newsletter/` | The **newsletter `.md` file** from your bot, plus any past issues. | Newsletter templating + on-site archive |
| `spreadsheets/` | The **Excel file** (calculator/formulas). | Formula audit + basis for the stamp-duty / pre-buying tools |
| `inspiration/` | Screenshots or a `links.txt` of sites you like (whitefox, luxhabitat, anything else). | Design direction |

## Naming
No strict rules — just drop files in. If you have multiple versions, keep the
newest or add `-old` to the ones to ignore.

## After you paste something
Tell Claude what you added (e.g. "logo and the excel are in") and it'll pick them
up — analyse the Excel, wire the logo into the design system, etc.
