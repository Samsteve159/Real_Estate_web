The four phases

Where each phase stands

Phase 1

Scaffold + design the microsite

Done

Teaches: setup, design — a polished site at localhost with real listings.

  • Vite + React + TS + Tailwind front-end; Hono + Anthropic SDK API (Node-local, Cloudflare-ready)
  • Distinctive "inner-west editorial" design system — bone / navy / teal / terracotta
  • Seed data: listings, suburbs, recently-sold comps
  • 20 real listings scraped from his public site — pure HTML parsing, no LLM, $0
Phase 2

Instant Home Valuation

Done · live-tested

Teaches: Claude API, prompt engineering, structured JSON — address → range → seller lead saved.

  • Valuation form: suburb, street, type, bed / bath / car, land size
  • Claude reasons over comps → structured JSON: low, high, midpoint, confidence, rationale, comparables
  • Result UI labelled an "indicative estimate"; CTA drives to a real appraisal
  • Lead persisted to SQLite (name, contact, address, estimate)
Live test: Newport 3-bed → $1.18M–$1.32M, high confidence, cited a same-street comp; the seller lead row landed in data/leads.db.
Phase 3

AI Buyer Concierge Chatbot

Done · live-tested

Teaches: system prompts, tool use, light RAG, streaming — chat → matches → inspection booked → lead.

  • Floating chat widget with streamed responses + suggestion chips
  • System prompt scoped to Manifest RE; listings injected as context (light RAG)
  • Tool use: search_listings() (NL search) + capture_lead()
  • SSE token streaming to the UI
Live test: "3-beds under $1M near Newport" → search_listings ×2 → accurate matches; "inspect 31 The Avenue, I'm Jordan…" → capture_lead wrote the lead.
Phase 4

Polish + deploy

In progress

Teaches: deployment — hardening done; going live is the last mile.

  • Per-IP rate-limiting on valuation / chat / lead (429 + Retry-After)
  • In-memory analytics at /api/stats
  • Loading / empty / error states + global ErrorBoundary + 404 route
  • security-review — input size/numeric caps, CORS locked to an allowlist, stats gated by a token, email/NaN validation
  • code-review — caught & fixed a chat-history truncation bug
  • Final mobile QA on a real device
  • Set the Anthropic spend cap before sharing any link
  • Deploy: web → Cloudflare Pages · api → Workers · leads.db → D1
  • Package both tools as embeddable iframe / script widgets
